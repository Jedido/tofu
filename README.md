# tofu

## Project setup
1. install Node.js
2. run `npm install` in the base repository folder

### Compiles and hot-reloads for development
```
npm run serve
```

### Builds and deploys production
```
run_server.bat
```

### Starts dev server
```
run_devserver.bat
```

### Lints and fixes files
```
npm run lint
```
